    name

    getActionContext

    description

    This function generates a "context" object that provides a common interface for use by sync integrations.

    example

    const handler = async (session, context, card, request) => { const syncContext = context.sync.getActionContext(request.arguments.provider, context, request.context, context.privilegedSession) ) ..... }

    Parameters

    • provider: string

      The name of the integration e.g. 'github', 'discourse'

    • workerContext: WorkerContext

      Context object provided to action functions by the jellyfish worker

    • context: any

      Logging context object

    • session: string

      session token used to interact with Jellyfish
